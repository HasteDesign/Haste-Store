# Haste Store #

## Project Description ##

Welcome to Haste Store, a base-theme developed by [Haste Design](http://www.hastedesign.com.br/), to create awesome stores with WordPress and WooCommerce.

## Plugins Support ##

Haste Store supports natively these plugins:

* [WooCommerce](http://wordpress.org/extend/plugins/woocommerce/)
* [JetPack](http://wordpress.org/extend/plugins/jetpack/) Infinite Scroll
